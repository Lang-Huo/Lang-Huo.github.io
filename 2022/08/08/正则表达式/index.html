正则表达式

1. 基本字符

1. 字符

  • 确定字符
  • 匹配hello字符，hello字符就是确定的字符

hello曾小红 ,welcome to regex learn site

  • 不确定字符
  • 匹配验证码：\d{4},其中\d代表0~9数字

【阿里云】您正在登录验证，验证码2571，
切勿将验证码泄露于他人，本条验证码有效期15分钟。

  • 混合字符
  • 匹配体重数据：体重\d+KG,由确定字符和不确定字符组成

曾小红，男,举重运动员，28岁，体重56KG，身高175cm。最好成绩：挺举98KG

*编写正则时，可以先把确定字符的部分先写出来然后再往里面填写不确定部分

2. 范围字符

字符 描述
.任意字符 除换行符外的任意单个字符
\w单词字符 包括字符、数字、下划线, 大写\W表示非单词字符
\d数字 0~9任意单个数字，大写\D表示非数字字符
\s空白字符 包含：空格、tab制表符、换行符，大写\S非空白字符

3. 自定义范围字符

用[]表示

字符 描述
[abc123] 任意范围 一组由确定字符组成的字符集，取其中一个
[0-9] 数字区间范围 指定数字范围集，[0-9] 等同于\d
[a-z] 字母区间范围 指定字母范围集。
[\d\w\s] 固定范围集 由多个固定范围集，组成一个更大的集
[0-9A-F\s] 混合集 上述四个自定义混合组成的集
[^ ] 排除范围 表示[ ]集之外的任单个字符，如[^0-9] 表示非数字，效果等同于\D

4. 转义字符

正则中有些特殊字符，如果要匹配特殊字符本身，就要使用转义字符对特殊字符\进行转义。

特殊字符：[.$^]|

转义字符：\

2.逻辑控制符

1.基本逻辑控制

逻辑符 描述
` ` 或
() 子表达式 用于独立计算括号中的内容。如：`(张
{} 数量控制 对字符或字表达式的数量范围进行限定。如：张.{1,3} 表示“张”后面只能跟1到3个任意字符。

|或：作用于字符或子表达式中。

()子表达式：把一段逻辑分开独立运算，然后再组合在一起，可以嵌套使用，()还有一个功能就是用来分组，将括号里面包裹起来的分成一个组。

{}数量控制：作用于字符或子表达式中。

{n} 指定数量 匹配n个字符
{n,m} 指定范围 匹配n至m个字符，即至少n个，最多m个。
{n,} n个及以上 匹配n个或者n个以上字符
* 任意个 相当于{0,} 0个或无数个
+ 至少1个 相当于 {1,}
？ 0或1个 相当于 {0,1}

？同时表示最少话匹配(懒惰匹配)只能用在量词后面，但是量词不是固定词

3.分组

反向引用

匹配下面合法的标题：<(h(1-6])>.*?<\/\1>

反向引用：反向引用指通过\组号引用之前的分组，可以把分组理解成一个变量，在通过变量名(组号)引用。它不能引用之后的内容，固作称作反向引用

引用替换

替换操作指将正则匹配到的内容，替换成指定字符串，该字符串可通过$组号引用组进行拼装。通过$0可以引用整个匹配的内容。

前者是使用\组号，而替换使用$组号。

分组的其他应用

  1. (?<名称> )命名分组：这种命名组只能用于在程序中提取操作，不能进行反向引用，也不能用在替换操作中
  2. (?: ) 移除分组：如果只想用作子表达式，而不想用于分组就可以使用(?: )从分组列表中移除
  3. ( ( ) )嵌套分组：其组号的命名顺序是以开括号出现顺序为准。
  4. ()+分组使用量词：同一个分组如果使用了量词，该分组会代表多个值，这时通过$组号去提取值的时候会得到该组最后匹配的值

4.边界断言

边界断言：一段表达式前后满足指定条件该条件由一个表达式组成，基于前后字符能否匹配来计算布尔值。因断言部分不会消耗匹配字符，所以又叫零宽断言。

(?= ...) 前置断言 判定一段表达式前面是否满足条件
(?! ...) 前置否定断言 判定一段表达式前面是否不满足条件
(?<= ...) 后置断言 判定一段表达式后面是否满足条件
(?<! ...) 后置否定断言 判定一段表达式后面是否不满足条件

条件组合：

  1. h(?=条件1)(?=条件2) 且运算,指必须同时满足多个条件
  2. h(?=(?=条件1)(?=条件2)) 且运算,条件1与条件2组成一个新条件,并且两个条件都必须满足
  3. h(?=(?=条件1)|(?!条件2)) 或运算,满足条件1或者 不满足 条件2
  4. h(?=表达式1|(?=条件1)(?!条件2)) 混合运算,表示满足表达式1或者同时满足条件1、2

5.特殊边界断言

正则中用^ 表示文本的开头

^ 文本开头 默认情况下，表示断言文本开头
$文本结尾 默认情况下，表示断言文本结尾
^ 行开头 被m修饰后，^表示行的开头，等价于`(?<=\n
$ 行结尾 被m修饰后，$表示行的结尾,等价于`(?=\n
\A 文本开头 断言文本开头
\Z 文本结尾 断言文本结尾
\b单词开头或结尾 断言单词开头或结尾
\B \b 否定式,非单词开头或结尾 ，等价于(?<!\b) 或(?!\b)

6.修饰符

修饰符 描述
global 全局匹配 默认只返回匹配到的第一个结果，加上g匹配全部可匹配结果
insensitive 勿略大小写 不区分大小写
multi line 行边界修饰符 此模式下^和$可以分别匹配行首和行尾，默认情况下只能匹配文本开头与结尾
single line **.**通配修饰符 此模式下.能匹配任意字符，包括换行符
extended 注释修饰符 忽略空白字符，可以多行书写，并使用#进行注释说明
